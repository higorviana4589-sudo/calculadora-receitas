# calculadora-receitas
Web app para gestão de custos de produção de receitas. Desenvolvido com Python, Streamlit e SQLite. Aplica conceitos de POO (Programação Orientada a Objetos) para organizar o cálculo de insumos, rendimento de porções e precificação automática. Projeto criado para auxiliar na gestão financeira de pequenos negócios.
# 🧮 Calculadora de Custos de Produção
Projeto desenvolvido para auxiliar na precificação de receitas de confeitaria.

## 🚀 Tecnologias Utilizadas
- Python (Lógica principal)
- SQLite (Banco de dados)
- Streamlit (Interface Web)

## 🛠️ Funcionalidades
- Cadastro de ingredientes e preços.
- Cálculo automático de custo por porção/unidade.
- Ajuste de margem de lucro.
